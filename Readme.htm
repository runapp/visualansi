簡介Visual Ansi

Visual Ansi是一款專門為bbs美工所設計的繪圖軟體，有別於目前一般的Ansi編輯器，VA提供更直覺滑鼠繪圖的方式來協助創作，並設置了動畫模式來簡化複雜的ANSI動畫製程，Visual Ansi 底下有兩個編輯器，一個是一般編輯器，用來製作一般的Ansi畫作，另一個則是方塊圖編輯器，方塊圖編輯器是利用一格四色的原理產生較小格的方塊圖，一般編輯器與方塊圖編輯器可以透過windows剪貼簿配合使用。

常見問題 Q&A

Q1.無法執行Visual Ansi
A: 通常是因為缺少系統檔或是VA本身檔案不完全所引起的問題

除了從錯誤訊息來判斷外 執行VA 環境測試程式 可以更明確的判定問題出在哪裡

[症狀一]. 找不到重要的.DLL檔-MSVBVM60.DLL ，VA 環境測試程式無法執行並出現同樣錯誤訊息
診斷: 表示你沒有VB Runtime(Visual Basic 6.0 Runtime )， 通常 Win Me 2000 XP 都會內建有著個檔案，Win98的使用者就必須要自己去下載

[症狀二].找不到裝置 或 找不到重要的檔案 xxxx.dll 或 xxx.ocx，VA 環境測試程式顯示你有缺少檔案
診斷:這表示你缺少某些系統檔，這時候你就必須用完整的安裝檔安裝一次

[症狀三].找不到裝置 或 找不到檔案，VA 環境測試程式顯示你沒有缺少系統檔
診斷:VA本身的檔案有缺失或是損毀，這裡指的檔案包含了Ansi.txt VACLoader.exe VisualAnsi.exe，這時請重新安裝

Q2.方塊圖貼到bbs上後會亂掉
A:一般bbs每行有字數的限制，像是PTT是一行只能容忍一定的位元數(注意中文字與許多特殊符號是雙位元)，多出來的就會自動斷行或是變成亂碼，這時候要想辦法減肥彩色碼或是去掉一部分的圖
這裡提供一些技巧 ，可以讓產生出來的彩色碼比較不會有這些問題
[技巧一].留空格
寬度不要完全填滿到最後，像預設的寬度不會超過79就是為此所做的調整
[技巧二].減少雙色字原理的出現(這個有點複雜，需要時再看，我自己寫完頭都暈暈的@@)
方塊圖的基本原理就是一格四色 ，而一格四色是雙色字的應用，雙色字會把一個字拆開中間加彩色碼，所以如果能減少這種情形發生就可以了，在實際的做法上，以橫觀(X軸;寬)來看同樣顏色而且連在一起的方塊盡量是偶數個，以縱觀(Y軸;高)來看，第奇數格和在他下面的那一格盡量是相同的顏色，像是(1,1)跟(2,1)，這個原理在於一個字元的高度是方塊的兩倍，所以在編譯成ansi文字後像這種的情形會變成空格然後用背景色，就不用用到這個"▄"字元，因而減少一格四色出現的機率


Q3.動畫貼到bbs上後有幾頁會亂掉
A:通常是因為bbs本身的問題所產生的，內容太多的動畫再貼上的時候常常會遭遇這個問題，像是PTT的話大約在第九十幾頁的地方會亂掉，解決的方法有兩個
[方法一].寄信+暫存檔
從別的可以正常貼上的BBS將內容寄到目標bbs的信箱，然後將整篇存入暫存檔，在你要的地方從暫存檔貼上
[ 方法二].分次貼上+暫存檔
這裡以PTT的環境為例
先到bbs中一個你可以輸入文章的環境(如寫信或po文..po文可以到test版)
然後先貼上第一部份後， 按下Ctrl+X 就會出現有關暫存檔的選項 (R/W/D)讀寫刪暫存檔？
輸入W來寫入暫存檔 這時他會要你選擇暫存0~9 隨便選一個記住就好
這時候如果你選的暫存檔已經有資料就會出現 暫存檔已有資料 (A)附加 (W)覆寫 (Q)取消？[A]
因為是貼上第一個部分可以選W覆寫， 這樣暫存檔裡就有你第一部份的資料了
然後這時候又回到編輯文字的地方， 你可以把之前第一部份的刪了或是直接Ctrl+X 選A取消然後再回到編輯文字的地方 如果上一個部分的內容很多 那還是用直接離開在重新的方法比較快
接下來做的動作跟第一部份一樣 ，只是再貼上後 寫暫存檔的地方 暫存檔已有資料 (A)附加 (W)覆寫 (Q)取消？[A] 要選A來附加 這樣就可以把第二部分接到第一部份後面
接下來重覆上面的動作直到你把所有的內容都存到暫存檔裡

到你想要貼上的地方(BBS可編輯文字情況下) 按Ctrl+X 輸入R來讀取存在暫存檔裡的內容
選擇你之前存的那個暫存檔就可以把所有的內容貼到你想要的地方了


Q4.畫布太大時工具屬性欄不見了
A:這是因為整個VA編輯器的大小超出了你的螢幕，所以剛好工具屬性欄就看不到了，這個時後你可以到工具(T)->選項 來調整字型的大小，把字調小就可以了，不過並不建議你這樣做，因為字太小的話字本身會變形，另一個方法是調整你螢幕的大小，方法是在桌面按右鍵選內容->設定 ，然後把螢幕區域調大(1024x768像素以上)這樣就可以了

Q5.有功能不知道怎麼使用
A: 請參閱 VA使用手冊 ，如果看過仍有不明白的請聯絡我，將改進於使用手冊的內容或本站的Q&A
